{% blocktrans %} The leaderboard now uses the Sudoku Power Index (SPI) — a score from 0 to 100 that rewards fast solves on difficult puzzles and consistent performance as you complete more puzzles. {% endblocktrans %}

{% trans "Per‑puzzle contribution" %}

{% blocktrans %} Each time you solve a puzzle, you earn a score \( S_{u,s} \) that reflects your relative performance: {% endblocktrans %}

\[\displaystyle S_{u,s} = \bar{t}_s \cdot \frac{\bar{t}_s}{t_{u,s}}\]

{% blocktrans %} This value increases if you solve the puzzle faster than average — and especially so if it's a tough puzzle. {% endblocktrans %}

{% trans "Symbol" %} {% trans "Meaning" %}
\( \bar{t}_s \) {% trans "Average solve time for puzzle \(s\)" %}
\( t_{u,s} \) {% trans "Your time to solve puzzle \(s\)" %}
{% trans "User score" %}

{% blocktrans %} Your score \( R_u \) is now the sum of your scores from the last 100 puzzles you’ve completed. If you have solved fewer than 100 puzzles, then only those contribute. This design means that building a top score requires completing many puzzles. {% endblocktrans %}

\[\displaystyle R_u = \sum_{s=1}^{N_u} S_{u,s}\]

{% blocktrans %} This total rewards players for consistent participation over many puzzles. {% endblocktrans %}

{% trans "Leaderboard score" %}

{% blocktrans %} To compare across players, each score \( R_u \) is normalized so that the top score always corresponds to an SPI of 100: {% endblocktrans %}

\[\displaystyle \text{SPI}_u = 100 \cdot \frac{R_u}{\max\limits_v R_v}\]

{% blocktrans %} The closer your performance is to the best, the closer your SPI will be to 100. {% endblocktrans %}
